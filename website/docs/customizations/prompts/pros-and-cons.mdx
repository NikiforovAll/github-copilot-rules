# Pros and Cons Analysis Prompt

A structured prompt for evaluating solutions with scored dimensions, side-by-side comparison, and a clear recommendation.

## Overview

This prompt enables GitHub Copilot to:

- Analyze proposed solutions for strengths and weaknesses
- Score solutions across relevant dimensions (complexity, performance, scalability, maintainability, security, cost)
- Compare alternatives in a side-by-side table
- Highlight best practices and common pitfalls
- Provide a concrete recommendation with trade-off justification
