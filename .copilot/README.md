# .copilot — VS Code User-Level Prompts

Files in this directory are synced to `~/AppData/Roaming/Code/User/prompts/` via chezmoi.

```bash
./co-chezmoi.sh apply    # sync repo → VS Code
./co-chezmoi.sh diff     # preview changes
./co-chezmoi.sh re-add   # pull changes from VS Code
./co-chezmoi.sh status   # show managed files
```
